flixGURU

A graphical guide to this week's top grossing movies

Bubble Size

The size of each movies bubbles is determined by the aggregate criticalscore calculated by Rotten Tomatoes.The scores range from 0-100 (0 being an awful movie and 100 being great. FlixGURU takes this score and scales the bubbles accordingly therefore thelarger a bubble reperesent more critically acclaimed movies.

This is not a Rotten Tomatoes website. It only scrapes the top 30 movies at the box office in the past week according to Rotten Tomatoes and displays them in an interesting way. This project uses the Vishna project as it's d3 model.
